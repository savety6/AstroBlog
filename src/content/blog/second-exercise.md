# Beginner's Guide to MongoDB Compass

![MongoDB Compass](/assets/images/compassBoy.webp)

## Introduction
MongoDB Compass is a GUI (Graphical User Interface) tool provided by MongoDB to help developers and database administrators visually interact with their databases. It simplifies data visualization, query building, and database management without requiring command-line operations.

## Why Use MongoDB Compass?
- **User-friendly Interface**: Manage databases with an intuitive GUI.
- **Visual Data Exploration**: View collections and documents in an organized manner.
- **Schema Analysis**: Automatically detect and display schema structures.
- **Query Building**: Easily construct and test queries using a visual editor.
- **Aggregation Pipeline**: Build complex data aggregation queries step by step.
- **Index Management**: Monitor and manage indexes for performance optimization.

## Installation
### Windows, macOS, Linux
1. Visit the official [MongoDB Compass download page](https://www.mongodb.com/try/download/compass).
2. Choose the appropriate version for your operating system.
3. Install the application following the setup instructions.

## Connecting to a MongoDB Database
1. **Open MongoDB Compass**.
![MongoDB Compass](/assets/screenshots/upr2/Compass_first_screen.png)
2. **Click on "Add new connection"**:
   - If running MongoDB locally, the default url is: `mongodb://localhost:27017`
   - If it's clean install you can press **"Save and Connect"** to establish a connection.
   
![MongoDB Connect modal](/assets/screenshots/upr2/Compass_new_connection.png)

## Navigating the Interface
![MongoDB Compass](/assets/screenshots/upr2/compass.png)
### 1. **Home Dashboard**
   - Displays recent connections.
   - Allows quick access to databases.

### 2. **Databases & Collections**
   - View all available databases and their collections.
   - Click on a collection to see its documents.
![MongoDB Compass](/assets/screenshots/upr2/Databases_Collections.png)
   - click the **"+"** and add this:
   ![MongoDB Compass](/assets/screenshots/upr2/compass_crate_database.png) 

### 3. **Documents Tab**
   - Browse documents in a JSON-like format.
   - Edit, delete, or insert new documents.

### 4. **Schema Tab**
   - Analyze the structure of documents.
   - Detect field types and relationships.

### 5. **Query Builder**
   - Construct queries using a simple UI.
   - Filter, sort, and project fields without writing raw MongoDB queries.

### 6. **Aggregation Pipeline**
   - Build complex queries using aggregation stages.
   - Preview results in real-time.

### Querying Data
1. Go to the **Documents** tab.
2. Use the query bar to filter results.
   Example:
   ```json
   { "age": { "$gt": 25 } }
   ```
3. Click **"Find"** to retrieve matching documents.

### Updating a Document
1. Select a document.
2. Click **"Edit"** and modify the fields.
3. Click **"Update"** to save changes.

### Deleting a Document
1. Select a document.
2. Click **"Delete"** and confirm removal.
